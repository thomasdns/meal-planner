# Meal Planner

Application web de planification de repas hebdomadaire.

## Objectif

Ce projet a pour but de construire une application full stack avec Next.js, TypeScript, PostgreSQL, Prisma et NextAuth.

## Fonctionnalités prévues

- Inscription et connexion
- Gestion des recettes
- Catégories de recettes
- Planning hebdomadaire
- Génération de liste de courses
- Tableau de bord
- Profil utilisateur

## Stack technique

- Next.js
- TypeScript
- Tailwind CSS
- PostgreSQL
- Prisma
- NextAuth
- Git / GitHub
- Vercel
